Este programa se ejecuta desde el modulo main.py que se encuentra dentro del directorio Taller5
El modulo main realiza la distribución completa de los demas modulos.

****** El programa realiza todo lo solicitado por el taller.

- Revisar clave de al menos 8 caracteres
- Contraseña contenga numeros y letras
- Contraseña sin letras o numeros repetidos
- Verifica que el USERNAME no exista en el json
- El correo debe tener '@'
- USERNAME y PASSWORD elimina espacios en blanco ya que no son permitidos
- El registro crea a un nuevo jugador en el json
- El login Implementa los tres mensajes (Usuario Correcto / Usuario No existe / Credenciales invalidos)

****** Extras

Se implementan views

OTRAS NOTAS:
El programa no ocupa instalar nada adicional, ya que lo que se necesita se utiliza como importaciones
